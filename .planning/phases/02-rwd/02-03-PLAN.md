**Aspect Ratio Strategy:**
- `aspect-ratio` property locks an element to a specific width:height ratio
- Combined with `object-fit: cover`, the image fills the container while maintaining ratio
- Prevents layout shift when images load (visual stability)
- Enables consistent proportions across all viewport sizes

**Typical Image Aspect Ratios by Category:**

| Category | Aspect Ratio | Usage | Comment |
|----------|--------------|-------|---------|
| **Hero Section** | 16/9 or 4/3 | Page headers, large backgrounds | Check current hero images in pages 0, 1, 2 |
| **Profile Cards** | 1/1 (square) | Clerk cards (page 3), menu avatars | Staff photos typically square |
| **Menu Items** | 3/2 or 4/3 | Page 4 menu images, Page 5 menus | Horizontal photos |
| **Gallery Cards** | 1/1 or 16/9 | Page 6 moments, Page 8 environment | Depends on original photos |

**Implementation Pattern:**
```css
.image-container {
  aspect-ratio: 16 / 9;      /* or 1 / 1, 4 / 3, etc. */
  overflow: hidden;
  
  img {
    width: 100%;
    height: 100%;
    object-fit: cover;        /* fills container while maintaining aspect ratio */
    object-position: center;  /* center crop if image is different aspect ratio */
  }
}
```
